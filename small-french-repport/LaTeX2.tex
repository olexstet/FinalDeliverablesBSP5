\section{Introduction}
 Le Bachelor Semestre Projet se concentrera sur le d\'eveloppement et le d\'eploiement d'une application web, appel\'e Jobs Observer, sur un serveur cloud AWS. Le \guillemotleft Jobs Observer \guillemotright \space utilises un logiciel existant qui permet de cr\'eer des statistiques en se basant sur les emplois dans le domaine du Science Informatique.  De plus une recherche dans le domaine du test sera effectu\'ee pour d\'emontrer qu'un programme est correcte.  

\section{D\'escription du projet}
Le premier objectif consiste \`a d\'emontrer qu'un programme est correct. Pour cela deux d\'emonstrations sont d\'evelopp\'ees. La premi\`ere permet de d\'emontrer qu'un programme est correcte de point de vu de l'ex\'ecution. La deuxi\`eme d\'emonstration se base sur les comportements pr\'ed\'efinis pour un programme qui doit les satisfaire.    
\newline
Le deuxi\`eme objectif est de d\'evelopper une application web qui sera d\'eploy\'e sur un serveur AWS, en utilisant \guillemotleft Django framework \guillemotright \space \'ecrit en python. L'application web a pour but de permettre \`a un utilisateur d'afficher ou de g\'en\'erer une statistique choisie. 


\section{Production}
La production finale du projet est une application web (r\'ef 5.3) et les deux d\'emonstrations (r\'ef 5.1 et 5.2). 
\newline
La premi\`ere d\'emonstration permet de d\'emontrer qu'un programme est correcte du point de vu de l'ex\'ecution. Tout d'abord, un programme est divis\'e en multiple bloque qui peuvent \^etre consid\'er\'ees comme des fonctions ou des parties individuelles d'un programme.  Apr\`es, quelques tests sont d\'efinis et r\'eunies dans un ensemble. 
\newline
Puis, tous les tests dans l'ensemble sont appliqu\'es sur chaque bloque d'un programme. Dans l'ensemble des tests, il y a des tests qui ne peuvent pas \`etre appliqu\'es \`a un bloque. Pour r\'esoudre, ce probl\`eme, chaque test est valid\'e avant d'\^etre appliqu\'e. Ensuite, les tests valid\'es sont appliqu\'es sur le bloque. Si le r\'esultat d'un test est le m\^eme que le r\'esultat d'un programme, alors le test est r\'eussi. 
\newline
Si, tous les tests sont r\'eussis alors, le program est correct du point de vu de l'ex\'ecution. 
\newline
La deuxi\^eme d\'emonstration utilise le comportement pr\'ed\'efini pour un programme et la premi\`ere d\'emonstration. Tout d'abord, les comportements pour chaque bloque du programme sont \'etablies. Chaque comportement est compos\'e de deux fonctions qui permettent d'\'evaluer les entr\'ees et les sorties d'un programme. 
\newline
Pour \'evaluer qu'un comportement est correct, les entr\'ees et les sorties d'un programme et d'un test doivent \^etre dans ce comportement. De plus, le test appliqu\'e doit \^etre r\'eussi. Pour d\'emontrer que tous le programme est correct, tous les comportements doivent \^etre satisfait et tous les tests doivent \^etre r\'eussi.  
\newline
L'application web d\'evelopp\'ee, est compos\'ee de \guillemotleft Front-end \guillemotright \space et \guillemotleft Back-end \guillemotright \space. La partie \guillemotleft Front-end \guillemotright \space est l'interface utilisateur qui permet \`a un utilisateur d'interagir avec l'application. 
\newline
L'interface utilisateur est compos\'ee de plusieurs pages HTML comme \guillemotleft Home \guillemotright, \guillemotleft About Software \guillemotright, \guillemotleft Plots \guillemotright \space et \guillemotleft Statistic \guillemotright. En utilisant la page de \guillemotleft Plots \guillemotright, l'utlisateur pourra choisir une statistique qui sera affich\'ee sur une page \guillemotleft Statistic \guillemotright. 
\newline
La partie du \guillemotleft Back-end \guillemotright \space est utilis\'ee pour g\'en\'erer les statistiques \`a la demande, afficher la partie de \guillemotleft Front-end \guillemotright \space et de d\'eploy\'e sur un serveur. 


\section{Conclusion}
Dans ce projet, nous avons d\'emontrer comment un programme peut \^etre correct du point de vu de l'ex\'ecution et de comportement en utilisant le raisonnement logique et la d\'eduction naturelle. De plus, nous avons d\'evelopp\'e une application web qui permet d'afficher et de g\'en\'erer les statistiques.
